# bump-cli

Bumps semantic versions.

```sh
$ bump package.json component.json

package.json
  3  "version": "0.3.1",  /* was 0.3.0 */

component.json
  3  "version": "0.3.1",  /* was 0.3.0 */

proceed? [Yn]
```

## Install

```sh
$ npm install -g bump-cli
$ bump --help
```
